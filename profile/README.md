## Installation

**Linux** and **MacOS** - Install from Terminal to `/usr/local/bin`:

```shell
curl -fsSL https://nanobus.io/install.sh | /bin/bash
```

**Windows** - Install from Command Prompt:

```shell
powershell -Command "iwr -useb https://nanobus.io/install.ps1 | iex"
```
